# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

BITOO takes security seriously. If you discover a security vulnerability, please **do not** open a public issue.

Instead, send details to **khcharem.omar@gmail.com**.

We will:
1. Acknowledge receipt within 48 hours
2. Provide an estimated timeline for a fix
3. Notify you when the vulnerability is resolved

## Responsible Disclosure

We request that you:
- Give us reasonable time to fix the issue before disclosure
- Provide sufficient details to reproduce and understand the vulnerability
- Avoid exploiting the vulnerability or accessing unauthorized data

## Scope

This policy applies to the BITOO application and its dependencies. For vulnerabilities in upstream packages (Flutter, Dart packages), please report to the respective maintainers.
